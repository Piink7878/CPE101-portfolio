# CPE101 Portfolio

A personal portfolio website built with **Next.js 16**, **React 19**, **TypeScript**, and **Tailwind CSS**.

## Features

- Hero section with a bold landing layout
- About section with profile card and tech stack tags
- Academic Journey timeline
- Works section with responsive project cards
- Sticky navigation with section-based highlighting

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- ESLint

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Build

```bash
npm run build
npm start
```
